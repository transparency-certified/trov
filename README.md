# Transparent Research Object Vocabulary

This repository contains the source files for the Transparent Research Object
Vocabulary (TROV), an OWL ontology for describing transparent research objects
and the systems that create them.

TROV is part of the [TRACE project](https://transparency-certified.github.io/)
(TRAnsparency CErtified), which develops standards for certifying that
computational research results were produced as claimed.

## Links

- **TROV 0.1:** https://w3id.org/trace/trov/0.1
- **Pre-release:** https://w3id.org/trace/2023/05/trov
- **Specification:** https://transparency-certified.github.io/trace-specification/docs/specifications.html

Documentation is built and published via
[trace-vocabularies](https://github.com/transparency-certified/trace-vocabularies),
where this repo is included as a git submodule.
