TROV 0.1 DRAFT

This vocabulary is a draft and subject to revision. Please submit feedback or report issues.

The Transparent Research Object Vocabulary (TROV) provides terms for describing transparent research objects and the trusted research systems that create them. The following hand-curated documents are the authoritative references for TROV 0.1:

  • TROV Vocabulary Reference — term descriptions and cardinality constraints
  • TRO Declaration Format — structure and field reference
  • TROV Extension Guide — adding institution-specific terms
